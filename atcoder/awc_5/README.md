# 错题记录
# C
1. 需要想到将 $|B_i - B_{i+1}| \le K$ 这个条件转化为 $-K \le B_i - B_{i+1} \le K$
2. 是否能求出每个 $B_i$ 的下限
3. 问题变成经典的约束递推：每个位置都取到最优下限，后一个数也能递推取到:
- $B_{i + 1} \ge B_i - K$: 

  可令 $B_0=A_0$，然后对于 $i>0$，$B_i=\max(A_i, B_{i - 1} - K)$
- $B_i \ge B_{i + 1} - K$: 

  可令 $B_{n-1} = A_{n - 1}$，然后对于 $i < n - 1$，$B_i=\max(A_i, B_{i + 1} - K)$

对于每个位置 $i$ 两种情况取最大值即为最终的 $B_i$
